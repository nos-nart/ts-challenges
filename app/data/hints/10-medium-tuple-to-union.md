#### Hint 1: Indexed Access
**The Goal:** Convert the array's values into a union of types.

**The Nudge:** In JavaScript, you can access an array by index (`arr[0]`). In TypeScript, how do you get *all* the possible values of an array `T` at once?

#### Hint 2: Special Index
**The Goal:** Use the standard numeric index trick.

**The Nudge:** What happens if you index an array with the keyword `number`? Think about `T[number]`.
